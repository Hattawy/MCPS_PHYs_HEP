This directory is created by Mohammad Hattawy (mhattawy@anl.gov) on Thursday, March 23rd, 2017 to analyze the MCPs root files. 

- The input root files have to listed in Global_analyzer.h, line 209
- The Global_analyzer.C is the script file where the analysis is performed
- run.cc is the compiler: run it as "root -l run.cc"
- The output plots will be saved in the folder named "plots"
